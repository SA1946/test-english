## 🛠️ Installation

1️⃣ Clone the repository:

```bash
git clone https://github.com/SA1946/test-english.git
```
2️⃣ Navigate to project directory:

```bash
cd test-english
```
3️⃣ Run or Open with Live Server
```bash
Alt+L Alt+O
```
